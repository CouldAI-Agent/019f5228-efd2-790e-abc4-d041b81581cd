# JT Soluções e Comércio

Um aplicativo Flutter moderno para a **JT Soluções e Comércio**, oferecendo uma interface limpa e responsiva para apresentar os serviços de soluções tecnológicas e o comércio de equipamentos da empresa.

## Funcionalidades

- **Apresentação da Empresa:** Tela inicial com informações sobre a JT Soluções e Comércio.
- **Catálogo de Serviços:** Cards responsivos detalhando as áreas de atuação (Soluções em TI, Comércio de Equipamentos, Consultoria).
- **Interface Responsiva:** O layout adapta-se perfeitamente em dispositivos móveis (Android/iOS), Web e Desktop.

## Stack Tecnológica

- [Flutter](https://flutter.dev/) - Framework de UI multiplataforma
- Dart - Linguagem de programação

## Como Executar

1. Certifique-se de ter o Flutter instalado na sua máquina.
2. Clone o repositório.
3. Execute `flutter pub get` para baixar as dependências.
4. Execute `flutter run` para iniciar o aplicativo.
